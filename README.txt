# PDF Appraisal Viewer

## Setup
1. Open Command Prompt (cmd).
2. Go into the folder: cd PDFAppraisalViewer
3. Install dependencies:
   pip install -r requirements.txt
4. Run the app:
   streamlit run app.py

## Usage
- Upload a PDF file OR use the included `sample.pdf` to test quickly.
- Use the sidebar checkboxes to show/hide sections.
- If no checkbox is selected, the full report is displayed.
